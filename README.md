# next-base64
> Base64 for browerjs

## install:
```shell
npm install -S afeiship/next-base64 --registry=https://registry.npm.taobao.org
```

## usage:
```js
import NxBase64 from 'next-base64';

const encode_str = nx.Base64.encode('afei');
// YWZlaQ==
const decode_str = nx.Base64.decode(encode_str);
// afei
```
